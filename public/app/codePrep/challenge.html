Palindrome{{title}}

Description

Create a Palindrome function that will have one input paramter that is a string.
Return true or false if the string is a palindrome. {{description}}

Examples

For example if your Palindrome function is sent the string 'dad' then it should return true.
It should return false if passed the string 'foobar'.{{examples}}

Tests

Palindrome('dad') should return true
Palindrome('foobar') should return false
Palindrome('able was I ere I saw Elba') should return true{{tests}}
